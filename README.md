# Seminar Hall Booking System

A web-based system to manage seminar hall reservations in a college environment.

## Features

• User Registration and Login  
• Seminar Hall Booking (Full Day / Half Day)  
• Admin Approval Workflow  
• Booking Status Tracking  
• Hall Availability Display  
• Booking Cancellation  

## Tech Stack

Frontend:
HTML
CSS
Bootstrap
JavaScript

Backend:
PHP

Database:
MySQL

Server:
XAMPP

## System Architecture

Browser → Apache Server → PHP Backend → MySQL Database

## Database Schema

Users Table
Bookings Table
